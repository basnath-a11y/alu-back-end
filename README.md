alu-back-end API exercises

This folder contains solutions for API exercises (0-3) that fetch and export TODO data from JSONPlaceholder.

Usage:

- `python3 api/0-gather_data_from_an_API.py USER_ID`
- `python3 api/1-export_to_CSV.py USER_ID`
- `python3 api/2-export_to_JSON.py USER_ID`
- `python3 api/3-dictionary_of_list_of_dictionaries.py`
